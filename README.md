# Simple-Text-Editor
Goal of the website: To implement a simple Text Editor website through which users can download the content that they type and copy the typed content to clipboard.

Front-end Technologies Used: HTML, CSS, JS  
No Backend and Databases are used.  
Deployment: This project has been deployed on Netlify.  
Deployment Link: https://simple-text-editor-a1b4.netlify.app/  

Project Timeline: Jul 12, 2026  

# Features
1. Users can copy the content that they type to clipboard
2. Users can download the typed content into a text file
3. Users can view the word count and character count of the text that they have typed in the website.
4. Users can clear the text area by pressing the *Clear Text* button.
5. Users can view the source code of the website by pressing the *Source Code* button.

# Versions
*Ver 1.0.0*: This is the first version of the project. It contains all the features that are listed above.
